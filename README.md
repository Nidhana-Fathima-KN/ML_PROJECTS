**House Price Prdeiction using machine learning
This project focuses on predicting house prices using machine learning techniques.
The dataset includes features like house size, number of bedrooms, neighborhood quality, and lot size.
Exploratory Data Analysis (EDA) is used to understand patterns and relationships in the data.
The analysis shows that factors like square footage and location have a strong impact on house prices. 
Different models such as Linear Regression, Polynomial Regression, and Random Forest are used for prediction. 
Model comparison helps in selecting the best approach for accurate results. 
Overall, this project demonstrates how data analysis and machine learning can be used to estimate house prices effectively.
