## House Price Prdeiction using machine learning

This project focuses on predicting house prices using machine learning techniques.
The dataset includes features like house size, number of bedrooms, neighborhood quality, and lot size.
Exploratory Data Analysis (EDA) is used to understand patterns and relationships in the data.
The analysis shows that factors like square footage and location have a strong impact on house prices. 
Different models such as Linear Regression, Polynomial Regression, and Random Forest are used for prediction. 
Model comparison helps in selecting the best approach for accurate results. 
Overall, this project demonstrates how data analysis and machine learning can be used to estimate house prices effectively.

##2-CLASSIFICATION
## Heart_Disease_Risk_Prediction using machine learning

This project focuses on predicting heart disease risk using machine learning techniques. The dataset includes important health and lifestyle factors such as age, blood pressure, cholesterol, smoking, and physical activity. Exploratory Data Analysis (EDA) is used to understand patterns and identify key factors affecting heart disease. Multiple models like Logistic Regression, KNN, SVM, Decision Tree, Random Forest, and Naive Bayes are applied for prediction. The analysis shows that features like chest pain, high blood pressure, cholesterol, and family history strongly influence heart risk. Model comparison is performed to select the best-performing algorithm. Overall, this project demonstrates how machine learning can help in early detection of heart disease and support better healthcare decisions.
